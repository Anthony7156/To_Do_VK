# ToDo VK Mini App

Простое приложение для управления задачами для ВКонтакте.

## Функциональность

- ✅ Добавление новых задач
- ✅ Редактирование существующих задач (двойной клик)
- ✅ Удаление задач
- ✅ Отметка выполненных задач
- ✅ Фильтрация задач (все/активные/выполненные)
- ✅ Очистка выполненных задач
- ✅ Сохранение данных в VK Storage
- ✅ Автоматическая загрузка данных пользователя
- ✅ Адаптивный дизайн для мобильных устройств

Инструкция по использованию разработанного мини-приложения
1. Как открыть мини-приложение
Существует несколько способов запуска, в зависимости от того, где вы находитесь:
По прямой ссылке. Самый простой способ. Разработчик может отправить вам ссылку вида https://vk.com/appID (где ID — уникальный номер приложения). Просто перейдите по ней в браузере на компьютере или телефоне.
2. Первый запуск
Режим разработки. Если приложение еще не опубликовано и находится в разработке, разработчик мог включить для вас специальный "Режим разработки". В этом случае вы сможете пользоваться приложением, даже если оно скрыто от других пользователей.
3. Основные функции и взаимодействие
Интерфейс. Внешний вид мини-приложения, скорее всего, стилизован под дизайн ВКонтакте, так как большинство разработчиков используют библиотеку VKUI. Это значит, что кнопки, переключатели и меню будут выглядеть знакомо и интуитивно понятно.
