# ToDo VK Mini App

Простое приложение для управления задачами для ВКонтакте.

## Функциональность

- ✅ Добавление новых задач
- ✅ Редактирование существующих задач (двойной клик)
- ✅ Удаление задач
- ✅ Отметка выполненных задач
- ✅ Фильтрация задач (все/активные/выполненные)
- ✅ Очистка выполненных задач
- ✅ Сохранение данных в VK Storage
- ✅ Автоматическая загрузка данных пользователя
- ✅ Адаптивный дизайн для мобильных устройств

## Технологии

- React 18
- VK Bridge
- VKUI (опционально)
- CSS3
